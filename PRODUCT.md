# Product

## Register

product

## Users

Security-minded engineers, benchmark authors, and agent developers use Trust Me Bro while evaluating whether coding agents resist hidden instructions, unsafe tool use, credential traps, and canary-triggering workflows. They are scanning dense traces, comparing model behavior, and deciding whether an agent can be trusted in real development environments.

## Product Purpose

Trust Me Bro is a local-first agent security benchmark console. It lets users run safe virtual scenarios, inspect terminal and reasoning traces, compare model outcomes, generate new attack scenarios, and archive benchmark evidence without risking real secrets or host-side effects.

## Brand Personality

Forensic, exacting, and artificially analytical. The interface should feel like a high-trust analysis instrument: precise, quiet, technical, and intentional, with enough visual distinction to feel memorable rather than generic.

## Anti-references

Avoid generic SaaS dashboard cards, hacker-movie neon overload, glassmorphism, cartoon security imagery, marketing-page hero layouts, and decorative effects that reduce trace readability.

## Design Principles

1. Lead with evidence: traces, canary state, scores, and model comparisons should always feel more important than decoration.
2. Make risk legible at a glance: semantic color, icons, and status hierarchy should help users distinguish pass, warn, fail, blocked, and tripped states quickly.
3. Keep density disciplined: the console can be information-rich, but spacing, alignment, and typography must preserve scanability.
4. Practice safe evaluation: the UI should reinforce that scenarios are virtual, canaries are intentional, and external model spend is controlled.
5. Feel analytical, not theatrical: visual polish should suggest artificial analysis and instrument-grade precision without compromising standard product affordances.

## Accessibility & Inclusion

Target WCAG AA contrast for text and controls, preserve visible focus states, avoid relying on color alone for safety status, and respect reduced-motion preferences.
